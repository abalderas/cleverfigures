DESCRIPCIÓN DEL PROYECTO

CleverFigures es un analizador estadístico cuantitativo-cualitativo para wikis en entornos docentes.

DESCRIPCIÓN DEL ÁRBOL DE DIRECTORIOS DEL PROYECTO

El repositorio del proyecto sigue la estructura estándar de Subversion: 
	- Carpeta trunk para version actual.
	- Carpeta branches para desarrollos paralelos.
	- Carpeta tags para versiones oficiales estables.
	
La estructura de la carpeta trunk, con el desarrollo actual, sigue la estructura estándar de las aplicaciones de CodeIgniter (metodología MVC).
Prácticamente todos los archivos que forman la aplicación se encuentran en los directorios:
	- trunk/application/controllers
	- trunk/application/views
	- trunk/application/models

CÓMO OBTENER EL PROYECTO

Puede descargar el repositorio completo de CleverFigures con la siguiente orden:
	svn checkout https://forja.rediris.es/svn/cleverfigures
Si lo prefiere, puede descargar paquetes con las versiones estables en 
	http://forja.rediris.es/frs/?group_id=1181

REQUISITOS DEL PROYECTO

Para ejecutar CleverFigures necesitará tener un servidor donde alojarlo, ya que se trata de una aplicación web.
También necesitará PHP >= 5, MySQL y conexión a internet.

PASOS PARA REALIZAR LA INSTALACIÓN DEL PROYECTO EN UN SISTEMA

Para instalar CleverFigures solo debe crear una base de datos con permisos de escritura en MySQL,
descomprimir el paquete descargado en el directorio correspondiente de su servidor, y acceder al mismo a través del navegador.
El asistente de instalación de CleverFigures le guiará en el resto del proceso.
