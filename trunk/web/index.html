What is CleverFigures?

CleverFigures is a web-application intended to perform qualitative and numerical statistical analysis of WikiMedia systems in educational enviroments. It includes data analysis, filtration and visualization of reports via dynamic charts as well as integration of external qualitative and numerical sources.

It allows to make combined analysis of groups of pages, users, etc. to see how a collective has worked, as well as comparative analysis, in which we can see the differences among students' assignments. This project has been developed in PHP + CodeIgniter and it's licensed under GPL.
